## 9. Risks / Notes

- **Backward compatible**: All existing 85 tests pass unchanged; schemas without rules produce identical output.
- **nonempty() annotation omitted**: Unlike `min(n)`, `nonempty()` does not emit a `minLength` annotation because it would incorrectly overwrite a stricter `minLength` from a prior `min(n)` call during merge.
- **annotation merge**: Uses last-write-wins `map.set` for duplicate keys. This is correct for the constraint model (e.g., `int()` overrides `type: "number"` with `type: "integer"`).
- **Pre-existing changes NOT included**: `cmd/main/main.mbt`, `cmd/main/pkg.generated.mbti`, `examples/real_llm_agent/schemas.mbt` are user changes from before this session and are excluded from the commit.
